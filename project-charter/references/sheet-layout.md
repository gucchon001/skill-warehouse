## 背景・目的・成功状態の質（C5 / E5）

- **背景（C5）**: **【現状（AsIs）】** 今どうなっているか・何が課題か。**【あるべき姿（ToBe）】** このプロジェクトで実現したい状態。**【ギャップ・原因の仮説】** AsIs と ToBe の差と理由の仮説。
- **目的・成功状態（E5）**: **【卒業要件】** ここまでできたらこの PJC として完了。**【歓迎要件】** あればよいが必須ではない範囲。よくばりすぎない。
