# スキル貯蔵庫

個人用の **Agent Skills** を一か所に置き、Cursor / Claude Code / Antigravity など複数エージェントから同じライブラリを指すためのリポジトリです（ローカル正本は多くの環境で `~/.agent-skills`）。

**英語スラッグ（GitHub リポジトリ名の例）**: `skill-warehouse`  
旧称 `cursor-global-skills` は Cursor 専用の印象が強いため、**スキル貯蔵庫**という呼び方と上記スラッグに寄せています。

同期・ジャンクション・Git push の手順はスキル **`skills-multi-host-sync`**（同梱の `references/details.md`）を参照してください。
