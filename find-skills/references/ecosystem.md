# 公開 Agent Skills エコシステム（検索・取得）

ローカル `~/.cursor/skills` と `.cursor/skills` に無いときの補完用。内容は製品更新で変わりうるため、公式サイトを正とする。

## Web

- **ディレクトリ・ランキング**: [https://skills.sh/](https://skills.sh/)
- キーワードはサイト内検索またはカテゴリ閲覧で行う。

## CLI（例）

Skills 系 CLI は配布元によってコマンドが異なる。代表的なパターン:

```bash
npx skills find "<自然言語またはキーワード>"
```

利用可能なら、リポジトリ README に記載の `add` / `install` で取得する。**実行前に**パッケージ名・レジストリが信頼できるか確認する。

## 選定の目安

1. **タスクとの一致**: README / `description` の WHEN にユーザーの動詞・ドメインが含まれるか。
2. **メンテ**: 最終更新・Issue の活発さ。
3. **信頼**: 既知の組織・十分なスター・他スキルからの参照（あくまで参考指標）。

## ホスト間の配置

グローバルに入れたスキルを Cursor 以外でも使う場合、**skill-folder-spec.md §9.1** のとおり Claude Code / Antigravity 用パスへコピーまたはジャンクションで同期する。
